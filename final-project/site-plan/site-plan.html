Overview

Purpose

The purpose of this websiteis to have an easily-accessible and fast-loading knowledge base for the "travelogue" that I'm writing for my group's D&D campaign, called Deathless.

Audience

The audience for this website is primarily the other members of the campaign, but also friends/family/etc. who we might be excited to show it to.

Dynamic elements

Character bio pages will be dynamically generated, all using the same HTML and CSS files. Some elements of the home page may be dynamically generated (mainly the character mugs) and the travelogue page will have selectable filters to hide/show some elements.

Wireframes

Landing page

Title, introductory blurb, some page links, and "Meet the adventurers" character mugs (which are also page links to those characters' bios -- info is carried over using URL params)

Character Bio Page

The site uses a single page for displaying character bios, populated dynamically from JS objects. For the context of this project, these JS objects are hard-coded in a list in a separate js file. The page features checkbox filters for hiding/showing in-game dates and real-world session dates, as well as a dropdown menu for changing the text formatting to be less stylized (for accessibility).

Travelogue Page

This page writes out all the travelogue entries currently available, populated dynamically from JS objects. For the context of this project, these JS objects are hard-coded in a list in a separate js file. The page features checkbox filters for hiding/showing in-game dates and real-world session dates, as well as a dropdown menu for changing the text formatting to be less stylized (for accessibility).

Desktop Variants

The Desktop versions of pages will not be much different; certain elements will be laid out horizontally rather than vertically, and header menu icons on all pages will be replaced with the actual menu items as links at the top of the page.
